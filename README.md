# Multimodal RAG System for Financial Report Analysis

## Problem Statement

In the domain of financial analysis, professionals such as analysts and investors rely on annual reports and financial statements to understand company performance. These documents are usually provided in PDF format and contain a mix of text, tables, and charts.

A major problem is that these reports are very long and complex. Users find it difficult to quickly locate specific information such as revenue, profit, or financial trends. Traditional keyword search is not effective because important data is spread across multiple sections.

These documents are multimodal in nature:
- Text explains company performance
- Tables contain structured financial data
- Charts show trends and comparisons

To understand the full picture, users often need to look at all these formats together, which is time-consuming.

A Retrieval-Augmented Generation (RAG) system helps solve this problem by retrieving relevant parts of the document and generating answers based on them. This ensures that answers are accurate and based only on the document content.

By processing images and charts into text summaries, the system can search across all types of content.

The goal of this system is to allow users to ask questions in simple language and get accurate answers along with references such as page number and content type. This will make financial analysis faster and easier.
